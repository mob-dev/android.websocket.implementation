### What is this repository for? ###

* Websocket implementation.

### Dependencies

* okhttp3

### How do I get set up? ###

* Add okhttp3 dependency in app/build.gradle.
```
implementation 'com.squareup.okhttp3:okhttp:z.x.y.' // 3.11.0
```
* Create MainActivity class.
* See `initializeWebSocket` method.
